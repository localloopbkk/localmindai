# LayerMind AI / LocalLoop Website Pack

Static HTML website pack including:

- What is LayerMind AI?
- What is LocalLoop?
- LayerMind AI products section with LocalLoop listed as a core product
- Partners section featuring Participatory Citizen Lab
- Newsletter signup
- Early member application
- Project / community submission
- Partnership call-to-action and partner highlight
- Featured people/projects
- Product launch waitlist
- Press / exhibitions / publications
- Cybersecurity & trust section

## Files

- `index.html` — main site content
- `styles.css` — responsive visual design
- `script.js` — mobile navigation behavior

## How to use

Open `index.html` in a browser or deploy the folder to Netlify, Vercel, GitHub Pages, Cloudflare Pages, or your own server.

## Form setup

The form currently points to:

```html
https://formspree.io/f/your-form-id
```

Replace this with your own Formspree, Tally, Typeform, Airtable, Google Forms, or custom backend endpoint.

## Cybersecurity notes

Before launch, enable HTTPS, spam protection, admin access controls, secure storage for submissions, privacy policy, cookie policy if needed, and a process for reviewing public submissions.
